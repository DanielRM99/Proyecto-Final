# Proyecto Final - Mundo Otaku

## Descripción
Una página web donde poder compartir contenido de mangas, animes, noticias, etc...

### Secciones
- Manga - Daniel
- Anime - Cristóbal 
- Post/Noticias
- Autores
- Productos/Merchandising
- Videojuegos - Natalia

### Datos 
- Book 
    - Id: string,
    - Author: string,
    - Price: number,
    - Year: date 

- Videogame
    - Id: string,
    - Name: string,
    - Plataforma: string,
    - Year: date,
    - Price: number,
    - Image: string
